# Plugin Concepts

These files are design blueprints for future Noctalia plugins.

They are intentionally not listed in `registry.json` yet. The goal is to keep the source registry clean while defining implementation scope, UI direction, dependencies, IPC surface, and merge strategy before writing production QML.

Current concepts:

- `nclipper.md`: notes + clipboard convergence roadmap
- `nkeyflow.md`: keybind visibility and action routing layer for Niri/Hyprland
